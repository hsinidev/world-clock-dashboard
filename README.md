# 🌌 Doodax World Clock Dashboard

Welcome to the **Doodax World Clock Dashboard**, a high-performance, SEO-optimized web application designed to track real-time global timestamps across major cities. Set against a mesmerizing, animated nebula background, this tool combines aesthetic elegance with technical precision, making it the ultimate utility for remote teams, travelers, and time enthusiasts.

## 🌟 Key Features

-   **Precision Timekeeping**: Leverages the browser's native `Intl` API for millisecond-accurate, timezone-aware clock rendering without external API latency.
-   **Immersive Galaxy UI**: Features a custom-built, multi-layered CSS animation system simulating a deep-space nebula and starfield with multi-colored particle effects.
-   **SEO Pillar Content**: Integrated with a massive, schema-rich article on Global Time Synchronization to drive organic search traffic, featuring an optimized "Read More" toggle.
-   **Fully Responsive**: Specific breakpoints ensure a seamless experience from 4K desktops to mobile phones.
-   **Compliance Ready**: Includes dedicated modal interfaces for Privacy Policy, Terms of Service, and DMCA compliance to satisfy Google Policy requirements.
-   **Zero-Dependency Logic**: Built for offline capability and maximum speed.

## 🛠️ Tech Stack

-   **Frontend**: [React 18+](https://react.dev/)

-   **Styling**: [Tailwind CSS](https://tailwindcss.com/)
-   **Animation**: CSS3 Keyframes & Transitions
-   **SEO**: JSON-LD Schema & Semantic HTML5
